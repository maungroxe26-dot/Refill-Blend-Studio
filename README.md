# 🧪 Refill Blend Studio

Aplikasi berbasis web untuk membantu usaha parfum refill mengelola inventaris bibit parfum, menghitung rasio pencampuran (*layering*), serta mengkalkulasi Harga Pokok Penjualan (HPP) per botol secara otomatis dan akurat.

## ✨ Fitur Utama
- **Kalkulator Formula Dinamis**: Memungkinkan pencampuran beberapa bibit parfum dan pelarut (Absolute/Fixative) sekaligus.
- **Rasio Komposisi Otomatis**: Menampilkan persentase perbandingan antara Bibit vs Pelarut secara *real-time*.
- **Kalkulator HPP Akurat**: Menghitung modal bersih per botol berdasarkan harga bahan per ml ditambah biaya operasional kemasan/botol.

## 🛠️ Teknologi yang Digunakan
- Python 3
- Streamlit (Frontend & Backend Logic)
- Pandas (Data Management)
